----------------

- <h3>Librairies</h3>

You must install :

  - Colorama : write this line in the cmd -> <code>pip install colorama</code>

- <h3>Executable</h3>

Choose a language : English or French.

Download and unzip the "pendu_python.exe" or "hangmans_game_python.exe"

Launch the file.
